# Statements

Statements are operations that control program execution.
Unlike expressions, statements do not necessarily produce a value.

In `nat20` statements include assignments, function definitions, and the control flow statements.

#### Assignment statement
```nat
x = 27;
b = %q(banana);
```

#### Function definition
```nat
fun sum_up_roll(die, modifier)
  return die + modifier;
nuf  
```
